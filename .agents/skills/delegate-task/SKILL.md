# Delegate Task

Use OpenSpace when multi-step work should be delegated or coordinated.

1. Check whether the `openspace` MCP server is available.
2. If it is available, use OpenSpace tools to break the task into concrete subtasks and execute them.
3. Prefer local Codex tools for straightforward work that does not need delegation.
4. Keep the user informed about the delegation plan, active subtasks, and final outcome.

If the `openspace` MCP server is unavailable, continue with local tools and state that OpenSpace delegation could not be used.
