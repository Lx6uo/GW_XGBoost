# Skill Discovery

Use OpenSpace to find or evolve skills when the current skill set is insufficient.

1. Check whether the `openspace` MCP server is available.
2. Search for relevant skills in the configured local skill directories first.
3. If cloud access is enabled, expand the search to the OpenSpace community only when needed.
4. Evaluate the candidate skill against the user's task before using or installing it.
5. Tell the user what skill was selected, added, or updated.

If the `openspace` MCP server is unavailable, continue with the existing local skills and note the limitation.
